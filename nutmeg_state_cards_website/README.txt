Nutmeg State Cards & Collectibles Website

How to edit:
1. Open index.html in a browser to preview.
2. Open index.html in a text editor to change words, links, and inventory.
3. Open styles.css to change colors and design.
4. Replace logo.png with your final logo if needed.
